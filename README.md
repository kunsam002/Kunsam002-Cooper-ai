## Prerequisites
You need to following installed on your machine:
* Nodejs 20
* PNPM `npm install --global pnpm`

## Development

Run the dev server:

```shell
# Install dependencies
pnpm install

# Run Remix dev server
pnpm run dev
```

Tests:
```shell
# Vitest
pnpm run test
```
